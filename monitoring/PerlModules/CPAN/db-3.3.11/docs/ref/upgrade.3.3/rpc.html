Release 3.3: DB_ENV->set_server

The DB_ENV->set_server interface has been deprecated and replaced with the DB_ENV->set_rpc_server function. The DB_ENV->set_server interface will be removed in a future release, and so applications using it should convert. The DB_ENV->set_server interface can be easily converted to the DB_ENV->set_rpc_server function by changing the name, and specifying a NULL for the added argument, second in the argument list.
