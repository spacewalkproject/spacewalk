 40      """  
 41      Convert a query with named parameters (i.e. :id, :name, etc) into one 
 42      that uses %(id)s, %(name)s parameters instead. 
 43   
 44      python-psycopg2 requires parameters to be in this form, so to keep our 
 45      existing queries intact we'll convert them when provided to the  
 46      postgresql driver. 
 47   
 48      RETURNS: tuple with: 
 49          - the new query with parameters replaced 
 50          - hash of each named parameter to an ordered list of the positions 
 51            where it was used. 
 52          - number of arguments found and replaced 
 53      """ 
 54      log_debug(6, "Converting query for PostgreSQL: %s" % query) 
 55      new_query = re.sub(r'(\W):(\w+)', r'\1%(\2)s', query.replace('%', '%%')) 
 56      log_debug(6, "New query: %s" % new_query) 
 57      return new_query
