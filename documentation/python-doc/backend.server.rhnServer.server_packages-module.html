update_errata_cache(server_id)

source code 

Function that updates rhnServerNeededPackageCache by deltas (as opposed to calling queue_server which removes the old entries and inserts new ones). It now also updates rhnServerNeededErrataCache, but as the entries there are a subset of rhnServerNeededPackageCache's entries, it still gives statistics regarding only rhnServerNeededPackageCache.
